# mba-server-standing-dock
This is a standing dock for a MacBook Air 2017. however it may work with similar thicknesses. The structure is 2-part, which allows it to be scaled to any width of computer. This is primarily used for running a home server on the MacBook in clamshell mode, which the openness of the dock provides good ventilation.
